# 人事管理系统 java551

## 项目概述

人事管理系统（简称：微人事）采用前后端分离的架构设计，以 Spring Boot 和 Vue.js 为主要技术栈，旨在提供高效、便捷的人事管理解决方案。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

- **职位部门管理**：实现公司职位和部门的增删改查等操作。
- **薪资管理**：对员工薪资进行管理，包括薪资结构设定和薪资发放等。

## 系统角色

- 系统管理员：负责系统维护、权限分配等。
- 普通用户：执行日常的人事操作，如信息录入、查询等。

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：现代浏览器

## 部署步骤

1. 准备运行环境：安装 Java 和 MySQL。
2. 导入数据库脚本。
3. 部署后端服务：运行 Spring Boot 应用。
4. 部署前端服务：将 Vue.js 项目构建后的文件部署到 Web 服务器。

## 目录结构

```
project/
│
├── backend/    # 后端代码
└── frontend/   # 前端代码
```

## 核心亮点

- **前后端分离**：使得系统更易于维护和扩展。
- **高效的事务处理**：利用 Spring Boot 的强大性能，高效处理人事相关业务。
